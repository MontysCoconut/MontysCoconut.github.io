Try Monty

Welcome to Try Monty. This is an interactive learning environment for the Monty programming language. Monty was designed to be readable and easy to use. You can find some more information about the language on the project's website (http://informatik.uni-bremen.de/monty).

Instructions

Description and Tasks:
The left side of the window (this panel) contains instructions and description of the tasks. Here you will find explainations of the language concepts and tasks to solve.
Monty Code Editor:
The upper right panel contains a code editor for Monty code. You may use it to try out some little programs. To execute the program, you can type moin into the terminal below the editor. Keyboard shortcuts for executing the code are CTRL+ENTER or ALT+ENTER.
The Terminal:
The terminal supports several commands. The command next can be used to go to the next page. Similarly the command prev can be used to go one step back. To clear the terminal, the command clear can be used. moin executes the code inside the editor.

Either type next into the terminal or click the ""-button to show an overview of all available challenges.
