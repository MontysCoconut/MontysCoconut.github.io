Try Monty

Try Monty is a learning platform for the Monty programming language, which was developed within a one-year student project at the University of Bremen. The project was realized by 14 computer science students.

The idea of the learning platform Try Monty was developed by Christopher Nottrodt and Carsten Pfeffer and is inspired by the interactive Ruby tutorial Try Ruby. The implementation of Try Monty and the Moin Monty interpreter was done by Carsten Pfeffer.

More information about the language can be found at the project's website at http://informatik.uni-bremen.de/monty. You may also want to get the code from GitHub .
