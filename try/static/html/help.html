Description and Tasks:
The left side of the window contains instructions and description of the tasks. Here you will find explainations of the language concepts and tasks to solve.
Monty Code Editor:
The upper right panel contains a code editor for Monty code. You may use it to try out some little programs. To execute the program, you can type moin into the terminal below the editor. Keyboard shortcuts for executing the code are CTRL+ENTER or ALT+ENTER.
The Terminal:
The terminal supports several commands. The command next can be used to go to the next page. Similarly the command prev can be used to go one step back. To clear the terminal, the command clear can be used. moin executes the code inside the editor.
Navigation:
You can navigate through the challenges by typing commands into the terminal, or by clicking the navigation buttons at the top and bottom of the left panel. The buttons may be deactivated when a next or previous step is not available. In that case you either want to follow the instructions on the left panel or return to the list of available challenges by clicking on the icon in the navigation bar.
